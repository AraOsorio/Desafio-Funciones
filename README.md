# Desafío 3 – Funciones en JavaScript

## Descripción
Este proyecto corresponde al desarrollo de ejercicios prácticos enfocados en la transformación y uso de funciones en JavaScript. Se aplican conceptos como funciones de expresión, arrow functions y manejo de eventos del DOM, con el objetivo de reforzar habilidades fundamentales mediante la modificación y creación de código interactivo.

## Contenidos trabajados
- Conversión de funciones tradicionales a funciones de expresión.
- Implementación de arrow functions de una línea.
- Separación de archivos HTML y JavaScript.
- Uso de `addEventListener` para eventos de click y teclado.
- Manipulación dinámica de estilos en el DOM.
- Creación de elementos HTML mediante JavaScript.

## Estructura del proyecto
- `1_funcion.js`: transformación de función declarativa a función de expresión.
- `2_arrow.js`: conversión a arrow function.
- `pintar.html` y `assets/js/script.js`: interacción para cambiar colores mediante eventos.
- `4_colores.html`: eventos de click y teclado, variable global y creación dinámica de divs.

## Objetivo académico
Aplicar buenas prácticas de organización del código, separación de responsabilidades y uso de funciones modernas en JavaScript para desarrollar interacciones básicas en páginas web.
